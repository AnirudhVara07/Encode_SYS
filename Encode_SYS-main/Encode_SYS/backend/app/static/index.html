Vigil (rule-based automation)

Uses the same Vigil template signals as overnight learning on hourly candles. Multiple strategies vote: on each tick, a strategy counts as a buy edge when its signal becomes BUY (was not BUY before), and a sell edge when it becomes SELL. If buy edges > sell edges, Vigil buys buy USD; if sell edges > buy edges, it sells sell fraction of your BTC. Manual Buy/Sell still work. Stop Vigil before saving config.
